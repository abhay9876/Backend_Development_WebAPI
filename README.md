# Backend_Development_WebAPI

## Description
This repository is dedicated to mastering **Backend Development using .NET Web API**, covering everything from REST fundamentals to advanced backend architecture.  
It provides hands-on implementation to help developers understand how to **design, build, secure, and optimize scalable APIs** for real-world applications.

---

## Tech Stack
- **Framework:** ASP.NET Core Web API  
- **Language:** C#  
- **Database:** SQL Server  
- **ORM:** Entity Framework Core  
- **Caching:** Redis  
- **Logging:** Serilog  
- **API Testing:** Postman  
- **API Documentation:** Swagger  
- **Version Control:** Git & GitHub  

---

## Repository Structure & Branches

### `main` Branch
Contains the base Web API setup and project initialization.

---

### `webapi-review` Branch
Complete implementation of a real-world **Task Manager Backend API**:
- CRUD operations (Create, Read, Update, Delete)
- RESTful API design
- Proper routing and controller structure
- Request & Response handling
- Integration with frontend

**Purpose:**
> To build a production-style backend API with clean architecture and real-world use cases.

---

### `fundooNotes` Branch
Backend support for Fundoo Notes application:
- Notes management APIs  
- User-based operations  
- API integration with Angular frontend  

**Purpose:**
> To implement backend services for a full-stack application.
